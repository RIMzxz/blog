Belajar HTML,CSS,DAN PHP

Apa itu HTML?

HTML digunakan untuk menentukan elemen-elemen di halaman web, seperti: Judul: Menentukan judul halaman. Paragraf: Menampilkan teks dalam bentuk paragraf. Gambar: Menyisipkan gambar ke dalam halaman. Tautan: Membuat link ke halaman lain atau situs web lain.

Apa itu CSS?

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan gaya halaman web. Jika HTML adalah kerangka rumah, maka CSS adalah cat, CSS digunakan untuk: Mengubah warna teks dan latar belakang. Mengatur ukuran dan jenis font. Menentukan tata letak elemen di halaman. Menambahkan efek visual seperti bayangan dan transisi.

Apa itu PHP?

PHP adalah bahasa pemrograman yang digunakan untuk membuat halaman web dinamis. PHP digunakan untuk: Mengelola data dari formulir (seperti formulir kontak). Menghubungkan ke database untuk menyimpan dan mengambil data. Membuat halaman web yang dapat berubah berdasarkan interaksi pengguna (misalnya, menampilkan nama pengguna setelah login).
